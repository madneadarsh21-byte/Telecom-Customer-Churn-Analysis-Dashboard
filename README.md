# Telecom Customer Churn Analysis Dashboard

# Project Overview

This project focuses on analyzing customer churn patterns in the telecom industry using Microsoft Excel. The dashboard helps identify factors influencing customer churn and provides actionable insights for improving customer retention strategies.

# Objectives

* Analyze telecom customer churn trends
* Identify high-risk customer segments
* Monitor customer retention metrics
* Build an interactive Excel dashboard for business insights

# Tools & Technologies

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Data Cleaning & Transformation

# Data Cleaning Process

* Removed inconsistencies and formatting issues
* Created helper columns for churn analysis
* Converted categorical values into analysis-friendly format
* Checked missing and duplicate values
* Organized data for pivot analysis

# KPI Metrics

* Total Customers
* Churn Customers
* Churn Rate %
* Average Account Length
* Average Service Calls

# Dashboard Features

* Interactive slicers and filters
* State-wise churn analysis
* Customer service calls vs churn analysis
* International plan churn analysis
* Voice mail plan churn analysis
* Account length vs churn analysis

# Key Insights

* Customers with international plans have higher churn probability.
* Higher customer service calls indicate increased churn risk.
* Long-term customers are more loyal.
* Certain states show significantly higher churn rates.

# Business Recommendations

* Improve customer support quality
* Introduce loyalty and retention programs
* Target high-risk customers with personalized offers
* Improve international plan pricing and services

# Project Outcome

The dashboard provides a clear understanding of customer churn behavior and supports data-driven business decisions for customer retention and service improvement.
